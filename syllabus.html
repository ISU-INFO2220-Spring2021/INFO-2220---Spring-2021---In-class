INFO 2220 Syllabus

Course Description

Introduction to interactive web development using a client-side language like JavaScript. Basic programming concepts common to almost all programming languages form the basis of the course. Exercises are designed to enhance students' problem-solving techniques and analytical thinking skills.

WEB SITES

Moodle will used to distribute the class materials, and announcements. All assignments will be turned electronically in Moodle. All content will be available on Moodle before the appropriate class period begins.

ZERO GRADES

If the instructor finds an assignment that has been submitted as your own work and is not, it will be recorded as a 0. If you are found cheating on exam, your exam will be taken and your result for the exam will be recorded as a 0.

Submit your own work.

HOMEWORK

Homework assignments are due Monday by 11PM the week following the appropriate lecture of the material. Do the homework, it helps.

Late homework will forfeit 25% of the grade if it is overdue for the first 24 hours and 100% after that.

EXAMS You will be tested on material from lecture, textbook, and assignments. Any information covered in class is considered for the exams. The exams will be given at the beginning of the designated class period. The comprehensive final will be based upon the material covered in the lecture, textbook, question sets, and assignments. If a student is unable to sit for the midterm exam, he/she must contact the instructor two weeks prior to the exam date and make alternate arrangements. Failure to follow this procedure will result in a zero for that exam. Students are expected to be available during finals’ week to take the final exam, changes to the final exam time will only be permitted if another class has a final at the same time as our final. The exam will be rescheduled for another time during finals’ week.

LEARNING OBJECTIVES

By the end of the course the student will be able to:

  • Design and implement interactive web pages.
  • Explain and utilize a wide range of the features available in the JavaScript programming language.
  • Write complex interactive scripts using JavaScript.
  • Analyze programming requirements in order to understand what data and processes are involved in the system.
  • Design a modular approach to satisfy those requirements.
  • Organize program code in web pages to implement the design.
  • Verify that the results obtained satisfy the original requirements.
  • Understand and implement JSON web sources.
  • Understand the underlying principles of a prototyped language and the prototyping chain.
